Follow the below steps in order to run the code:


1) First of all install python3 from the link "https://www.python.org/downloads/"

2) This project runs on the flask server, therefore you need flask on your machine to run the project,
   Use the following command to install the flask framework,
	pip install Flask
	Read the quickstart documentation from the following link(http://flask.pocoo.org/docs/0.12/quickstart/) to have a knowledge on flask server
3) Install the dependencies by using command if required,
	pip install pandas_datareader
	pip install datetime
	pip install fix_yahoo_finance 
	pip install numpy
	pip install matplotlib

4) Run the project on the command prompt using below command,
	Python main.py
        Copy the URL " http://127.0.0.1:5000/" and Paste in the browser (This project is tested on Browsers; Google Chrome and Mozilla Firefox)
	OR
	pip install -t lib -r requirements.txt
        python main.py

Important Note: User must have a working internet connection to fetch real time data from yahoo finance
